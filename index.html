Project Overview

Background

KAORIUM operates AI-powered fragrance discovery consoles across 400+ retail locations in Japan. As they expand to the UK market, they need a simple CRM to manage customer data captured during fragrance experiences and enable targeted email marketing based on scent preferences.

Current Deployments 400+ locations in Japan
Data Location Japanese server (GDPR compliant)
MVP Timeline 6-8 weeks

MVP Objectives

  • Enable multi-tenant authentication for retail partners
  • Integrate with Japanese server for customer data
  • Provide segmentation by date, fragrance preferences, and descriptive words
  • Create simple email interface with scheduling capabilities
  • Ensure GDPR compliance and data security

MVP Functional Requirements

Authentication System

  • Email and password authentication for each client
  • Dedicated interface per authenticated client
  • Secure multi-tenant data isolation
  • Role-based access control

Segmentation by Date/Period

  • Extract emails registered within specific date ranges
  • Filter by interaction date and time
  • Support for custom date range selection

By 'Most Liked' Fragrance

  • Extract emails for users who selected a specific fragrance as most liked
  • Filter by single fragrance selection
  • Support for multiple fragrance queries

By 'Liked' Fragrances

  • Extract emails for users who included a specific fragrance in their top 3
  • Support for ANY or ALL matching logic
  • Combine with other filters

By Selected Words

  • Extract emails for users who selected specific descriptive words
  • Filter by any of the 3 words per product
  • Word frequency analysis

Email Interface

  • Email composition interface with rich text editor
  • Clickable hyperlinks in message body
  • Schedule sending time functionality
  • Send to segmented lists from features 1-4
  • Email preview before sending

Data Structure

interaction_date DateTime Date and time of KAORIUM experience
email String User's email address
liked_fragrances Array[3] 3 fragrances tapped as 'liked'
most_liked_fragrance String 1 fragrance selected as 'most liked'
descriptive_words Array[9] 3 words per fragrance (total 9)
